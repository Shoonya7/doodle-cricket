How to Add This Cricket Game to Your Website

Follow these simple steps to embed the ABP Cricket game on your own website:

Step 1: Add the HTML Element

Add this div element where you want the game to appear:

<div id="shoonya-cricket"></div>

Step 2: Add Required CSS and JavaScript

Add these links in your HTML head or body sections:

<!-- In the head section -->
<link rel="stylesheet" crossorigin href="https://cricket-abp.vercel.app/assets/index.css">

<!-- Before closing body tag -->
<script type="module" crossorigin src="https://cricket-abp.vercel.app/assets/index.js"></script>

Step 3: That's it!

The game will automatically initialize and display in the specified div.
